🚀 DÉMARRAGE RAPIDE
─────────────────────

WINDOWS (PowerShell ou CMD)
───────────────────────────
1. Installer Visual Studio Community (gratuit)
   https://visualstudio.microsoft.com/fr/

2. Ouvrir le dossier MiniArcade dans Visual Studio

3. Visual Studio détectera automatiquement CMake et proposera :
   "Générer le cache CMake" → Cliquez sur OUI

4. Une fois terminé :
   Menu → Déboguer → Exécuter sans débogage
   
   OU utilisez les commandes :
   cd MiniArcade
   mkdir build
   cd build
   cmake ..
   cmake --build . --config Release
   .\Release\MiniArcade.exe


LINUX (Ubuntu/Debian)
─────────────────────
sudo apt update
sudo apt install build-essential cmake libsfml-dev
cd MiniArcade
mkdir build && cd build
cmake ..
make
./MiniArcade


macOS
─────
brew install cmake sfml
cd MiniArcade
mkdir build && cd build
cmake ..
make
./MiniArcade

❓ QUESTIONS FRÉQUENTES
───────────────────────

Q: Je n'ai pas CMake installé
A: Téléchargez-le depuis https://cmake.org/download/
   OU utilisez Visual Studio qui l'inclut automatiquement

Q: Erreur "Cannot find SFML"
A: Sur Windows, CMake téléchargera SFML automatiquement (patience!)
   Sur Linux/Mac, installez SFML avec votre gestionnaire de paquets

Q: Le programme ne démarre pas
A: Assurez-vous de lancer depuis le terminal/console
   Le dossier assets/ doit être à côté de l'exécutable

Q: Erreur de police
A: Vérifiez que assets/font.ttf existe dans le dossier du projet

🎮 COMMENT JOUER
────────────────

Menu Principal :
  • Flèches Haut/Bas : Naviguer
  • Entrée : Sélectionner
  • Échap : Quitter

Les jeux :
  1. REFLEX GAME : Cliquez vite quand le carré devient vert !
  2. SNAKE GAME : Mangez des pommes avec les flèches
  3. PONG GAME : Joueur 1 (Z/S) vs Joueur 2 (Flèches)

Dans chaque jeu : Échap = Retour au menu
